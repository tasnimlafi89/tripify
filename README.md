# Tripify - Travel Planner

## Setup
1. Clone: `git clone https://github.com/votreuser/tripify.git`
2. Frontend: `cd frontend && flutter pub get`
3. Backend: `cd backend && mvn spring-boot:run`


## Modules
- Auth: Voir /backend/src/main/java/com/tripify/auth/
- Recherche: Flux via /api/search/pay
